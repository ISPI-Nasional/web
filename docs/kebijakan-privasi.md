# Kebijakan Privasi ISPI Nasional

*Terakhir diperbarui: 1 Februari 2024*

## Pendahuluan

Ikatan Surveyor Pengawas Indonesia (ISPI) Nasional ("kami", "ISPI") berkomitmen untuk melindungi privasi dan keamanan data pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda.

## Informasi yang Kami Kumpulkan

### Data Pribadi

Kami mengumpulkan informasi berikut saat Anda mendaftar sebagai anggota:

- âœ… **Identitas Pribadi**
  - Nama lengkap
  - NIK (Nomor Induk Kependudukan)
  - Tanggal dan tempat lahir
  - Jenis kelamin
  - Alamat lengkap
  - Nomor telepon/HP
  - Alamat email

- âœ… **Data Profesional**
  - Riwayat pendidikan
  - Pengalaman kerja

  - Foto formal
  - CV/Portfolio

- âœ… **Data Keuangan**

  - Riwayat transaksi iuran
  - Invoice dan kwitansi

### Data Otomatis

Kami juga mengumpulkan data secara otomatis:
- Alamat IP
- Browser dan device info
- Log aktivitas website
- Cookies

## Bagaimana Kami Menggunakan Data

### Tujuan Penggunaan

Data Anda kami gunakan untuk:

1. **Administrasi Keanggotaan**
   - Verifikasi identitas
   - Pembuatan Kartu Tanda Anggota (KTA)
   - Pengelolaan database member
   - Komunikasi organisasi

2. **Layanan Member**
   - Akses sistem online
   - Pendaftaran event/pelatihan
   - Sertifikasi kompetensi
   - Newsletter dan update

3. **Keuangan**
   - Proses pembayaran
   - Invoice dan pelaporan
   - Administrasi iuran

4. **Pengembangan Organisasi**
   - Analisis demografi anggota
   - Evaluasi program kerja
   - Perencanaan strategis

## Pembagian Informasi

### Pihak Ketiga yang Dapat Mengakses Data

Kami TIDAK AKAN menjual atau menyewakan data pribadi Anda. Namun, data dapat dibagikan kepada:

1. **Pengurus ISPI**
   - Pengurus Pusat, Wilayah, Cabang (untuk administrasi)

2. **Penyedia Layanan**
   - Payment gateway (untuk proses pembayaran)
   - Hosting provider (untuk keamanan data)
   - Email service (untuk komunikasi)

3. **Lembaga Resmi**
   - Kementerian PUPR (untuk data anggota surveyor)
   - Lembaga Sertifikasi Profesi (LSP)
   - Hanya jika diwajibkan oleh hukum

4. **Partner Kerjasama**
   - Dengan persetujuan Anda

## Hak Anda

Sebagai pemilik data, Anda berhak:

### 1. Akses Data
- Melihat data pribadi yang kami simpan
- Download data pribadi Anda

### 2. Koreksi Data
- Mengubah data yang tidak akurat
- Update informasi profil

### 3. Penghapusan Data
- Request penghapusan data (setelah resign keanggotaan)
- Catatan: Data finansial tetap tersimpan untuk audit

### 4. Pembatasan Penggunaan
- Membatasi penggunaan data untuk tujuan tertentu
- Opt-out dari newsletter/komunikasi marketing

### 5. Portabilitas Data
- Meminta data dalam format yang dapat dibaca mesin
- Transfer data ke organisasi lain (jika berlaku)

**Cara Mengajukan:** Email ke privacy@ispinasional.or.id dengan subjek "Data Privacy Request"

## Cookies

### Apa itu Cookies?

Cookies adalah file kecil yang disimpan di browser Anda untuk:
- Menyimpan preferensi
- Analisis penggunaan website
- Keamanan login

### Jenis Cookies yang Kami Gunakan

1. **Essential Cookies** (Wajib)
   - Login session
   - Keamanan

2. **Functional Cookies** (Opsional)
   - Remember me
   - Preferensi bahasa

3. **Analytics Cookies** (Opsional)
   - Google Analytics
   - Traffic analysis

**Kontrol:** Anda dapat menonaktifkan cookies melalui pengaturan browser, namun beberapa fitur mungkin tidak berfungsi.

## Retensi Data

### Berapa Lama Kami Menyimpan Data?

| Jenis Data | Periode Penyimpanan |
|------------|---------------------|
| Data anggota aktif | Selama keanggotaan aktif |
| Data anggota non-aktif | 5 tahun setelah resign |
| Data transaksi | 10 tahun (sesuai regulasi perpajakan) |
| Data sertifikasi | Permanen (untuk rekam jejak) |
| Log sistem | 2 tahun |

### Penghapusan Data

Data akan dihapus secara permanen setelah periode retensi berakhir, kecuali diwajibkan oleh hukum.

## Perubahan Kebijakan

Kami dapat mengubah Kebijakan Privasi ini sewaktu-waktu. Perubahan akan:
- Diumumkan di website
- Dikirim via email ke seluruh anggota
- Berlaku 30 hari setelah publikasi

## Dasar Hukum

Kebijakan ini sesuai dengan:
- UU No. 27 Tahun 2022 tentang Perlindungan Data Pribadi
- Peraturan Menkominfo terkait
- Best practice internasional (GDPR)
